	代码位置 com.zuoye.hive
	
	3.2、如果两次浏览之间的间隔超过30分钟，认为是两个不同的浏览时间；再求每个id浏览时长、步长
	
		解题思路：
		
		1.使用lag函数求出与上一次浏览时间的时间差分钟数，如果大于30分钟标记为1
		
		2.对相同id的数据，开窗对标记求sum，这样可以给不同标记时间的数据打上标记
		
		3.浏览时长需要将第一次的浏览的时间间隔改为0
		
		4.对id和标记groupby即可得出结果
